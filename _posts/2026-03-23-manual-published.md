# LunyScript Manual & Scripting API Published

The [LunyScript Manual](https://lunyscript.com/documentation/) is now available!

The manual pages are entirely human-generated 😉 and cover the most fundamental concepts. I will continuously update it.

## Roadmap

Development focus is on Unity initially.

-    Spring 2026: Feature development
-    Summer 2026: Alpha/Beta-Testing
-    Autumn 2026: v1.0 Release

Ports to Godot, Unreal, and/or other engines will not start before 2027.

## Development Update

In recent weeks I have shifted development focus to Unity. This allows me to move faster.

Coroutines and Variables were a focus in recent weeks. I consider these finished for the time being.
Then I integrated Input System with Action Maps and device pairing. Local Multiplayer is fully supported!
And collision/trigger event callbacks are also implemented. Sach concept/feature has its own sample scene.

Next I will focus on tighter editor integration. Being able to see and edit script variables, seeing how blocks execute.
Also passing Inspector values into scripts.

I'm hopeful that by May I will have a pre-alpha version that can be reviewed and played around with.
